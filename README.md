A responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript. It presents my profile, featured projects, skills, work experience, problem-solving profile, and contact details in a clean one-page layout.

## Preview

- Hero banner with profile photo on the right
- Side-by-side skills tags and call-to-action buttons
- Mobile-friendly navigation and responsive stacking
- Clean dark visual style with orange accent color

## Features

- Responsive hero section with profile photo
- Smooth scrolling navigation
- Mobile-friendly menu
- Featured projects section
- Skills and experience highlights
- Competitive programming / problem-solving cards
- Contact links and downloadable CV
- Scroll progress indicator
- Accessible skip link and keyboard-friendly navigation

## Deployment

This project is ready for GitHub Pages deployment.

Steps for GitHub Pages:

1. Push the repository to GitHub.
2. In your GitHub repository, go to `Settings` > `Pages`.
3. Set the source to the `main` branch and the root folder (`/`).
4. Save the settings and wait for the Pages URL to be generated.

The site opens directly from `index.html` at the repository root.

You can also host this portfolio on:

- GitHub Pages
- Netlify
- Vercel
- Any static hosting provider
